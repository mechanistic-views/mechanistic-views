# MIB: Mechanistic Interpretability Benchmark

- **URL**: https://openreview.net/forum?id=sSrOwve6vb
- **Year**: 2025
- **Focus**: Standardized causal localization evaluation

## Tracks
1. Circuit Localization — identifying model components critical for task performance
2. Causal Variable — featurizing hidden vectors using SAEs and DAS

## Key Finding
SAE-derived features underperform raw neuron representations on causal variable track.
Attribution and mask optimization outperform alternatives for circuit localization.
Supervised DAS surpasses other featurization methods.
