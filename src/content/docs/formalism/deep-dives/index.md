# Deep Dives

Full mathematical treatments of the formalisms used by the higher-commitment views. These pages assume graduate-level mathematics — linear algebra throughout, plus differential geometry for gauge quotients, algebraic topology for sheaf cohomology, and singularity theory for stratification.

Each page corresponds to an accessible overview in the parent [Formalisms](/formalism/) section. Start there if you want the intuition without the proofs.

## Pages

- [Grassmannian Geometry](grassmannian/) — principal angles, geodesic distance, transport maps, Fréchet mean, computational tractability. Underpins the [subspace view](/views/subspace/); parent page: [Grassmannian](/formalism/grassmannian/).
- [Gauge Quotients and Holonomy](gauge-holonomy/) — gauge symmetries, fiber bundle structure, connection choices, holonomy computation, the holonomy identity conjecture. Underpins the [structural view](/views/structural/); parent page: [Fiber bundle quotient](/formalism/fiber-bundle-quotient/).
- [Sheaf Cohomology](sheaves/) — cellular cosheaves on directed graphs, $H^0$ and $H^1$, the localizability obstruction, spectral sequences. Used by the [structural](/views/structural/) and [stratified](/views/stratified/) views for circuit localizability.
- [Stratification](stratification/) — Whitney conditions, mechanism space decomposition, Thom-Mather control data, the $\mathcal{M}_\infty$ stratum. Underpins the [stratified view](/views/stratified/); parent page: [Whitney Stratification](/formalism/stratification/).
