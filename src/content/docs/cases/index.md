# Cases

Each case study applies the mechanistic views framework to a concrete interpretability example, showing how multiple views combine to characterize the same phenomenon at different levels of description. The cases range from well-understood circuits (IOI, induction heads) to frontier phenomena (hallucination, self-knowledge) where the framework highlights what evidence is still missing.

- [Indirect Object Identification (IOI)](ioi/) — primarily exercises the [object](/views/object/), [role](/views/role/), and [subspace](/views/subspace/) views
- [Induction Heads](induction/) — primarily exercises the [object](/views/object/), [role](/views/role/), and [process](/views/process/) views
- [Grokking](grokking/) — primarily exercises the [process](/views/process/) and [subspace](/views/subspace/) views
- [Hallucination](hallucination/) — primarily exercises the [object](/views/object/), [role](/views/role/), and [subspace](/views/subspace/) views
- [Self-Knowledge](self-knowledge/) — primarily exercises the [subspace](/views/subspace/) view
